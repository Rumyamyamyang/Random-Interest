# Random-interest

## 동생이 하도 돈을 안갚아서 만든 프로젝트

#### 실행 과정
* 처음에 값을 받는다.
* 최대 이자값을 기준 날짜에 따라 받는다.
* 이자율이 랜덤으로 나온다. (0~10의 경우   5.3, 3.7같이 소수점 한자리까지 나옴)
* 해당 %만큼 계산해서 최종금액이 나온다
* 이유는 모르겠지만 계산이 완료되면 바로 꺼져서 아무런 의미도 없는것을 생성하게 만든다.
